# Jawad Shaikh - Portfolio

Premium, responsive portfolio website for Jawad Shaikh, a Senior Software
Developer based in Abu Dhabi, UAE.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run lint
npm run build
npm start
```

For production SEO URLs, set:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

The project is ready for deployment on Vercel. The resume is served from
`public/Jawad Shaikh.pdf`.
